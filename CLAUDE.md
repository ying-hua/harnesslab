# CLAUDE.md

Project-wide instructions for Claude Code working in this repo.

## Commit messages

All git commit messages must be written in English, regardless of what language the conversation with the user is in.

## Code and CLI text language

All text that ships with the code and can be seen by someone outside this conversation — code comments, docstrings, CLI descriptions/help text, console/error/log messages, and test descriptions/fixtures — must be written in English, regardless of what language the conversation with the user is in.

Exceptions (kept in Chinese on purpose, don't "fix" these):
- `CLAUDE.local.md` — local-only instructions, not checked into the repo.
- `README.zh-CN.md` / `AGENTS.zh-CN.md` — the deliberate Chinese translations paired with the English `README.md` / `AGENTS.md`.
